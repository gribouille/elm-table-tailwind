## Usage

Install the ELM package:

```
> elm install gribouille/elm-table-tailwind
```

## Examples

To start the examples in dev mode:
```
> cd examples
> npm install
> npm run server
> npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
